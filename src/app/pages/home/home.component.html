Esse é um teste para uma vaga de frontend para a empresa Mout's

Um crud de prefeitos, cidades e estados.

O projeto está dividido entre backend e frontend e uma das hospedagens que usei foi a Umbler (para ter acesso ao banco de dados MySQL), com uma degustação de 7 dias gratis, logo, esse projeto deve ser avaliado o quanto antes

Como trata-se de uma avaliação com pretensão ao frontend, então eu fiz a API meia boca, sem utilizar algum padrão de projeto pois queria ganhar tempo! Achei mais válido caprichar no front com o angular. O design também não tá dos melhores (por isso usei bootstrap, para ganhar velocidade), foquei mais na lógica mesmo.
